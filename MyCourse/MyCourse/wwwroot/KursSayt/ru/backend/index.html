О программе

Рассматриваете BackEnd как серьезный шаг к будущей профессии? Всегда хотели углублённо изучать Java? Этот курс рассчитан на подготовку разработчиков, которые будут экспертами как в алгоритмах, так и в работе с серверной частью приложений и сайтов. Студенты станут заниматься как отдельно, так и в парах и группах. На занятиях нет сухой теории — материал подается понятно и четко, у каждого студента впереди – практика и живые примеры.

Программа включает модуль «Алгоритмы», который преподают только в профильных вузах. В результате выпускник будет знать такие языки, фреймворки и их производные как Spring и Hibernate, получит знания по базам данных и сможет создавать полноценные серверные системы и проекты.

Вступительные требования

Возраст от 18 лет

Знание английского не ниже уровня Intermediate

Наличие персонального ноутбука или компьютера

Готовность к интенсивному обучению

По окончанию обучения будете

  1. Знать язык программирования Java и уметь с ним работать
  2. Знать Backend фреймворки Spring IoC, Spring Security, Hibernate, Spring Data
  3. Знать алгоритмы и уметь анализировать быстродействие программы
  4. Разрабатывать веб-интерфейсы и приложения, управлять наборами и базами данных, тестировать и масштабировать программы
  5. Организовать систему безопасности сайта или приложения
  6. Владеть навыками soft-skills, уметь презентовать себя и свой продукт

Программа

Модуль 1.Java Basics

Включает в себя изучение базовых принципов работы с языком программирования Java, основные принципы объектно ориентированного программирования, работу с классами, интерфейсами, коллекциями Set, Map, Queue, Stack. Так же будет рассмотрены лямбда выражения, обработка ошибок, работа с файлами.

Модуль 2.Алгоритмы

Алгоритмы – важнейший модуль, с которым имеют дело студенты технических вузов. Это основа компьютерных наук, без которой невозможно успешно программировать. Полученные знания будут служить сильной технической базой при дальнейшем освоении любых технологий. В процессе данного модуля студенты освоят основные алгоритмы и структуры данных, обычные и типизированные массивы, связные списки, стеки, очереди, ассоциативные массивы. Также студенты познакомятся с хэшированием, графами, DFS и BFS, динамическим программированием, деревьями, алгоритмами кэширования и т.д.

После окончания данного модуля студенты будут иметь четкое представление сложности алгоритма, будут в состоянии анализировать быстродействие программы.

Модуль 3.Java Web

Целью этого модуля является научить студентов создавать и запускать в работу серверную часть web приложения, создавать базу данных, настраивать взаимодействия между ее таблицами, регулировать получение и отправку запросов на сервер.

Студенты получат опыт в таких направлениях:

  • сборщик проектов Maven
  • Servlet API, Jettyserver
  • SQL запросы
  • создание схемы для базы данных
  • распределение уровней доступа
  • Servlet filters
  • cookies
  • аутентификация

Модуль 4.Java Frameworks, Multithreading

Мы будем изучать Java EE при написании полноценных серверных систем. Поработаем с фреймворками Spring IoC, Spring Security, Hibernate, Spring Data, базами данных SQL, NoSql, веб-серверами и многопоточностью для параллельного выполнения нескольких операций.

Разберем основы и принцип работы современных фреймворков для лучшего понимания существующих стандартов в индустрии. Студентов ждет парное программирование, ориентация на проекты, изучение стилистики кода, аутентификация и авторизация, cookies, создание онлайн-магазинов.

Воркшопы.Business Intelligence, Mobile Application

На протяжении всех модулей студенты смогут посещать воркшопы и участвовать в вебинарах от международных специалистов в области ИТ.

Воркшоп «Business Intelligence»

Студенты получат понимание о сфере «Business Intelligence»(BI) и аналитике данных, узнают о развитии бизнес-анализа и его использовании. Основная тема: технологии, идеи для реализации хранилищ данных, визуализация данных и будущее направления BI.

Чему научатся студенты:

  • Сформируют понимание о том, что такое бизнес-анализ
  • Получат знания о новых инструментах аналитики данных
  • Сгенерируют идеи для реализации хранилища данных
  • Обсудят перспективы изучения этой сферы

Воркшоп «Mobile Development на базе React Native»

React Native — фреймворк, который позволяет использовать один язык программирования и для Android, и для IOS в разработке нативных мобильных приложений.

На воркшопе будут рассмотрены такие темы:

  • React и React Native: главные отличия
  • Настройка окружения
  • Создание билда проекта для Android и IOS
  • Использование специальных инструментов
  • Дебаг приложения

Воркшоп «Node JS»

Node JS — программная платформа, которая позволяет использовать JavaScript в куда более широком спектре применений. Благодаря Node JS стало возможным, без изучения дополнительного BackEnd языка разрабатывать серверную часть приложений.

Охват тем мероприятия:

  • введение в платформу Node.js
  • настройка среды и инструментов для разработки
  • организация приложений, модульный подход
  • работа с консолью (терминалом)
  • создание простого веб-сервера, работа с http-запросами
  • обработка событий (особенности работы Event Loop)
  • фреймворк Express
  • размещение Node.js приложения на облачном сервисе Heroku

Soft skills, технический английский и подготовка к собеседованию

С середины курса студенты параллельно начнут заниматься развитием своих внутренних качеств и готовиться к презентации проектов.

Мы создадим условия полноценных собеседований, максимально приближенные к реальным.

Проработаем резюме каждого студента, поработаем над его умениями презентовать проект и свой бренд.

Финальный проект

Команды студентов будут работать над комплексным финальным проектом под еженедельным наблюдением преподавателя, ментора и курс-координатора.. После 2,5 месяцев сплоченной работы, от планирования до подготовки демо-версий, студентам предстоит презентация проекта.
