Java dili ilə bağlı 8 maraqlı fakt

Java – Sun Microsystems şirkəti tərəfindən hazırlanmış obyekt yönümlü bir proqramlaşdırma dilidir. Java tətbiqləri adətən Java virtual maşınının (JVM) işlədiyi xüsusi bir bay-koda çevrilir. Öz növbəsində, JVM hər hansı bir cihazda quraşdırıla bilər. Bu, Java proqramlarının kross-platformasını təmin edir.

Java çox populyar bir proqramlaşdırma dilidir. Əsasən unikal daşınma ilə əlaqəli texniki bir fenomen halına gəlib: Java tətbiqləri hər hansı bir cihazda və ya əməliyyat sistemində işləyir. Onun əsas keyfiyyətləri arxitekturadan müstəqillik, proqramların sadə, lakin etibarlı şəkildə icrası və təhlükəsizliyin təminatıdır.

Bu da Java haqqında 8 maraqlı fakt:

• ABŞ-dakı 10 kompüterdən 9-da virtual Java maşını quraşdırılıb və korporativ kompüterlərin 97%-də dil istifadə edilir.

• Əvvəlcə Java interaktiv TV üçün hazırlanmışdı. Lakin kabel televiziyası sənayesi onu istifadə etmək üçün kifayət qədər inkişaf etməmişdi.

• Dil James Gosling tərəfindən hazırlanmışdı və əvvəlcə adı Oak idi. Gosling ofisinin yaxınlığında böyüyən palıd ağacına həsr olunmuşdu. Sonralar Green adı ilə adlandırıldı. Daha sonra isə Java. Java müştəri tətbiqləri və server proqramlarını yazmaq üçün istifadə edilirdi. Öz son adı ilə dil Java qəhvə markasına borcludur. Buna görə rəsmi loqoda tüstülənən bir fincan qəhvə göstərilir. Java-nın inkişafını indi Oracle idarə edir. 2010-cu ildə o, Sun şirkətini aldı.

• Java developerin ildə orta əmək haqqı 83 975 $

• Dünyada təxminən 9 milyon Java proqramçısı var.

• Tiobe.com-a görə, Java ən populyar proqramlaşdırma dilləri arasında ikinci yeri tutur. JavaScript-ə birinci yer verilir.

• JUnit proqram təminatını vahid sınaqdan keçirmək üçün əsas texnologiya Java kitabxanasından – 5 proqramçıdan 4-ü istifadə edir. Jenkins ikinci yerdədir – ondan developerlərin 70%-i istifadə edir.

• Java simvolu olan Duke, sonradan Shrek kimi məşhur cizgi filmləri üzərində işləyən Joe Palrang tərəfindən yaradıldı.
